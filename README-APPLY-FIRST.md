# Manual ready-to-replace package

Copy the folders in this ZIP directly into the project root:

`E:\Dot Net Web Development\ProjectManagement`

Allow Windows to replace the six matching files while preserving the folder structure. Do not copy the wrapper folder itself into the project root.

After replacement, delete root and test-project `bin` and `obj` folders, reopen Visual Studio, and rebuild the complete solution.

No database migration is required.
